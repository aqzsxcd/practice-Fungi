## Хронология выполнения работ

### Этап 1. Подготовка окружения (1 час)

- Установка Visual Studio 2022
- Скачивание и настройка SFML
- Создание пустого проекта
- Настройка Include Directories, Library Directories и Additional Dependencies
- Копирование DLL-файлов SFML

### Этап 2. Создание базового каркаса (2 часа)

- Написание структуры `Okno`
- Создание главного цикла приложения
- Реализация базовой отрисовки фона
- Обработка события закрытия окна

### Этап 3. Создание окон (1.5 часа)

- Реализация создания новых окон по нажатию клавиши N
- Генерация случайных цветов для каждого окна
- Отрисовка прямоугольников (окон) на экране
- Добавление рамок для визуального разделения окон

### Этап 4. Добавление заголовков и кнопки закрытия (2 часа)

- Загрузка шрифта (`arial.ttf`)
- Отрисовка заголовка окна (верхняя полоса)
- Отрисовка текста с названием окна ("Окно 1", "Окно 2"...)
- Отрисовка кнопки X с красным фоном

### Этап 5. Реализация закрытия окон (1.5 часа)

- Определение области кнопки X
- Проверка клика мыши на попадание в кнопку
- Установка флага `zakryto = true`
- Удаление закрытых окон из векторов `okna` и `cveta`

### Этап 6. Реализация перетаскивания окон (3 часа)

- Запоминание смещения клика относительно угла окна
- Отслеживание движения мыши при зажатой кнопке
- Обновление координат `x` и `y` перетаскиваемого окна
- Поднятие активного окна наверх (перемещение в конец вектора)
- Исправление проблемы с очередностью обработки событий

### Этап 7. Модификация (2 часа)

- Добавление поля `vstryaskaTimer` в структуру `Okno`
- Реализация запуска анимации при клике по телу окна
- Добавление сдвига окна вправо-влево при отрисовке
- Настройка параметров анимации (12 кадров, сдвиг на ±8 пикселей)
- Отладка конфликтов с перетаскиванием

### Этап 8. Документирование (4 часа)

- Создание UML-диаграмм (классов, последовательности, состояний, блок-схемы)
- Написание технического руководства для начинающих
- Создание README.md в формате Markdown
- Копирование итогового кода в документацию

### Этап 9. Подготовка презентации (2 часа)

- Запись видео с демонстрацией работы программы
- Подготовка HTML-страницы на основе Markdown
- Создание финального отчёта

### Этап 10. Итоговая проверка (1 час)

- Компиляция и тестирование всех функций
- Исправление мелких ошибок
- Подготовка к сдаче

## Проблемы и их решения

| Проблема | Решение |
|----------|---------|
| SFML не видит заголовочные файлы | Настройка Include Directories в свойствах проекта |
| Ошибка LNK2005 (main already defined) | Исключение Default.cpp из сборки |
| DLL-файлы не найдены при запуске | Копирование .dll в папку с .exe |
| Перетаскивание ломалось после модификации | Сброс флагов peretaskivaetsya перед каждым кликом |
| Встряска срабатывала на верхнем окне, а не на кликнутом | Двухпроходная проверка: сначала поиск кликнутого окна, потом обработка |

## Заключение

В ходе выполнения проекта был создан работающий оконный менеджер на C++ с использованием библиотеки SFML. Реализованы все основные функции: создание, закрытие, перетаскивание окон. Добавлена авторская модификация — анимация встряски при клике по телу окна.

Проект соответствует требованиям задания, код написан с использованием русских идентификаторов переменных (транслитом), документация оформлена в формате Markdown и HTML.
